# 프로그래머스 미로탈출

[문제 링크](https://school.programmers.co.kr/learn/courses/30/lessons/159993)

## 접근
- bfs를 이용하여 미로를 이동한다
- 레버를 만나면 방문 처리를 해주고, 그 이후에 출구를 만나면 탈출한다

### 초기 코드
```
import java.util.ArrayDeque;
import java.util.Queue;

public class Solution {
    public int solution(String[] maps) {
        int answer = 0;

        int[] dx = {-1, 1, 0, 0};
        int[] dy = {0 ,0, -1, 1};

        int N = maps.length;
        int M = maps[0].length();
        int[] start = new int[2];
        boolean isOpened = false;
        boolean isExited = false;

        char[][] map = new char[N][M];
        for (int i = 0; i < N; i++) {
            for (int j = 0; j < M; j++) {
                map[i][j] = maps[i].charAt(j);
                if (map[i][j] == 'S') start = new int[] {i, j};
            }
        } //for - insert map

        Queue<int[]> queue = new ArrayDeque<>();
        queue.add(start);

        while (!queue.isEmpty()) {
            int[] cur = queue.poll();
            int cx = cur[0];
            int cy = cur[1];
            answer++;

            if (map[cx][cy] == 'L') isOpened = true;
            else if (isOpened && map[cur[0]][cur[1]] == 'E') {
                isExited = true;
                break;
            }

            for (int d = 0; d < 4; d++) {
                int nx = cx + dx[d];
                int ny = cy + dy[d];

                if (nx >= 0 && nx < N && ny >= 0 && ny < M && map[nx][ny] != 'X') {
                    queue.add(new int[] {nx, ny});
                }
            } //for - d
        } //while - bfs

        if (!isExited) answer = -1;

        return answer;
    }
}
```

### 문제점
- 레버를 찾기 전에 출구를 지나오는 것을 생각하여 방문 처리를 하지 않았는데 이는 메모리 초과로 이어짐
- `answer` 변수 값 증가 위치가 잘못됨
- 하나의 while문에서 레버와 출구를 동시에 찾으려고 하여 bfs 진행이 꼬임

## 개선
- bfs를 시작-레버, 레버-출구 이렇게 2번 돌리기

### 개선 코드
```
import java.util.ArrayDeque;
import java.util.Queue;

public class Solution {

    static int[] dx = {-1, 1, 0, 0};
    static int[] dy = {0 ,0, -1, 1};

    static int N, M;
    
    public int solution(String[] maps) {
        
        N = maps.length;
        M = maps[0].length();
        int[] start = new int[2];
        boolean[][] isVisited = new boolean[N][M];

        char[][] map = new char[N][M];
        for (int i = 0; i < N; i++) {
            for (int j = 0; j < M; j++) {
                map[i][j] = maps[i].charAt(j);
                if (map[i][j] == 'S') start = new int[] {i, j};
            }
        } //for - insert map
        
        int toLever = bfs(map, start, 'L');
        if (toLever == -1) return -1;
        
        int toExit = bfs(map, start, 'E');
        if (toExit == -1) return -1;

        return toLever + toExit;
    } //solution

    private int bfs(char[][] map, int[] start, char target) {
        Queue<int[]> queue = new ArrayDeque<>();
        boolean[][] isVisited = new boolean[N][M];

        queue.add(new int[] {start[0], start[1], 0});
        isVisited[start[0]][start[1]] = true;
        
        while (!queue.isEmpty()) {
            int[] cur = queue.poll();
            int cx = cur[0];
            int cy = cur[1];
            int dist = cur[2];
            
            if (map[cx][cy] == target) {
                start[0] = cx;
                start[1] = cy;
                return dist;
            } //if - 목표지점 도착

            for (int i = 0; i < 4; i++) {
                int nx = cx + dx[i];
                int ny = cy + dy[i];
                
                if (nx >= 0 && nx < N && ny >= 0 && ny < M &&
                    !isVisited[nx][ny] && map[nx][ny] != 'X') {
                    isVisited[nx][ny] = true;
                    queue.add(new int[] {nx, ny, dist+1});
                }
            } //for - 사방탐색
        } //while - bfs
        
        return -1;
    } //bfs
    
}
```

### 회고
- bfs를 두 번 돌릴 생각은 해보지 못했다. 하지만 문제에 따라 여러번 활용할 수 있다는 것을 배웠다.
