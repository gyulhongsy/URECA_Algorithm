# 프로그래머스 방문 길이

[문제 링크](https://school.programmers.co.kr/learn/courses/30/lessons/49994)

## 접근
- 캐릭터가 이동한 길의 방문 처리를 하기 위해 좌표를 2배로 늘림
- 이동은 두 칸, 방문 처리는 두 칸 사이의 좌표에 함
- 음수 인덱스를 없애기 위해 범위를 0~20으로 조정

### 정답 코드
```
class Solution {
    public int solution(String dirs) {
        int answer = 0;

        int[][] map = new int[21][21];
        boolean[][] visited = new boolean[21][21];

        int cx = 10, cy = 10;
        for (char dir : dirs.toCharArray()) {
            int nx = cx, ny = cy;
            
            if (dir == 'U') nx = cx + 2;
            else if (dir == 'D') nx = cx - 2;
            else if (dir == 'R') ny = cy + 2;
            else if (dir == 'L') ny = cy - 2;

            if (nx >= 0 && nx < 21 && ny >= 0 && ny < 21) {
                int bx = (cx+nx) / 2, by = (cy+ny) / 2;

                if (!visited[bx][by]) {
                    visited[bx][by] = true;
                    answer++;
                } //if - 처음 가는 길

                cx = nx;
                cy = ny;
            } //if - 범위 안쪽인지
        } //for - dirs

        return answer;
    }
}
```

### 회고
- 이전에 좌표 이동을 처리해야 하는 비슷한 문제를 풀어서 이동하는 좌표의 범위를 2배로 늘려 방문 처리 하는 방법을 바로 떠올릴 수 있었다.
- 좌표 이동 처리를 할 때는 범위를 2배 늘리는 접근 방식도 있다는 것을 꼭 기억해두어야 겠다.
- 현재 좌표와 방문 처리를 할 좌표, 이동할 좌표를 초기화하는 위치가 조금 헷갈렸다. 꼼꼼히 확인할 필요가 있었다.
