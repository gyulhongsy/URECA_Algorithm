# 백준 1181번 단어 정렬

## 문제
알파벳 소문자로 이루어진 N개의 단어가 들어오면 아래와 같은 조건에 따라 정렬하는 프로그램을 작성하시오. <br>
1. 길이가 짧은 것부터
2. 길이가 같으면 사전 순으로 <br>
단, 중복된 단어는 하나만 남기고 제거해야 한다.

## 입력
첫째 줄에 단어의 개수 N이 주어진다. (1 ≤ N ≤ 20,000) 둘째 줄부터 N개의 줄에 걸쳐 알파벳 소문자로 이루어진 단어가 한 줄에 하나씩 주어진다. 
주어지는 문자열의 길이는 50을 넘지 않는다.

## 출력
조건에 따라 정렬하여 단어들을 출력한다.

## 정답 코드 (java)
```
import java.util.ArrayList;
import java.util.Collections;
import java.util.HashSet;
import java.util.Scanner;
import java.util.Set;

public class Main {
	
	static int N;

	public static void main(String[] args) {
		Scanner scan = new Scanner(System.in);
		
		N = scan.nextInt();
		scan.nextLine();
		
		Set<String> set = new HashSet<>();		
		for (int i = 0; i < N; i++) {
			set.add(scan.nextLine());
		}
		scan.close();
		
		ArrayList<String> strs = new ArrayList<>(set);
		
		Collections.sort(strs, (a, b) -> {
			if (a.length() == b.length()) {
				return a.compareTo(b);
			}
			return a.length() - b.length();
		}); //sort

		for (String str : strs) {
			System.out.println(str);
		} // for - print
		
	} //main
}
```

## 회고
- 처음에는 중복 요소에 대한 생각을 하지 못해서 그냥  `ArrayList` 로만 생성하였다.
- 중복을 제거하기 위해 단어를 받을 때는 `HashSet` 으로, 이후에 다시 `ArrayList` 로 변환하여 정렬하였다.
- 정렬 기능을 재정의 하면서 Comparator를 람다식으로 구현하는 것을 복습할 수 있었다.
