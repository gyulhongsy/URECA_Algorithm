# 프로그래머스 삼각 달팽이

[문제 링크](https://school.programmers.co.kr/learn/courses/30/lessons/68645)

## 초기 접근
- 트리 자료구조에 값을 넣고 전위순회 하기
  - 트리에 값을 어떻게 넣을 것인지..
  - 이 방법은 오히려 더 힘들다.
  - 트리구조는 부모-자식 관계가 명확하게 아래로 뻗어나가는 구조일 때 유리함
- 2차원 배열에서 아래쪽, 오른쪽, 왼쪽 대각선 위쪽 세 방향으로 숫자를 채운다
  - 벽(혹은 이미 숫자가 있는 칸)을 만남 -> 방향 전환
  - 아래 방향 숫자 넣기: i만 증가
  - 오른쪽 방향 숫자 넣기: j만 증가
  - 왼쪽 대각선 위 방향 숫자 넣기: i--, j--
  - (0, 0) 칸을 만나면 종료
  - 2차원 배열을 돌면서 0이 아닌 경우에만 정답 배열에 넣기
 
### 초기 코드
```
class Solution {
    public int[] solution(int n) {
        int[][] nums = new int[n][n];
        int num = 1;

        for (int i = 0; i < n; i++) {
            nums[i][0] = num;
            num++;
        } //for - 아래 방향 값 넣기

        for (int j = 1; j < n; j++) {
            nums[n-1][j] = num;
            num++;
        } //for - 오른쪽 방향 값 넣기

        for (int i = n-2, j = n-2; i > 0 && j > 0; i--, j--) {
            nums[i][j] = num;
            num++;
        } //for - 대각선 왼쪽 위 방향 값 넣기
        
        int[] answer = new int[num];
        int k = 0;
        for (int i = 0; i < n; i++) {
            for (int j = 0; j < n; j++) {
                if (nums[i][j] != 0) answer[k++] = nums[i][j];
                else continue;
            }
        } //for - insert answer

        return answer;
    } //solution
}
```

### 문제점
- 숫자를 겉 부분만 채우고 안쪽으로는 채우지 않았다.

## 개선
- 삼각형의 모든 칸을 다 채울때까지 방향 전환과 숫자 채우기를 반복한다.

### 개선 코드
```
class Solution {
    public int[] solution(int n) {
        int[][] nums = new int[n][n];

        int row = -1;
        int col = 0;
        int num = 1;

        for (int i = 0; i < n; i++) {
            for (int j = i; j < n; j++) {
                if (i % 3 == 0) { //아래
                    row++;
                } else if (i % 3 == 1) { //오른쪽
                    col++;
                } else if (i % 3 == 2) { //대각선 왼쪽 위
                    row--;
                    col--;
                } //if ~ else

                nums[row][col] = num;
                num++;
            } //for - j
        } //for - 방향 전환 횟수

        int[] answer = new int[num-1];
        int k = 0;
        for (int i = 0; i < n; i++) {
            for (int j = 0; j < n; j++) {
                if (nums[i][j] != 0) answer[k++] = nums[i][j];
                else continue;
            }
        } //for - insert answer

        return answer;
    } //solution
}
```

### 회고
- n의 값이 곧 방향을 전환하는 횟수라는 것을 깨닫는 것이 중요했다.
  - 전진하는 횟수가 0이 되면 반복문을 빠져나오면 된다.
- 따로따로 썼던 3개의 for문을 하나로 합쳐 조건에 따라 전진하는 방향을 정해준 뒤 값을 넣어주니 되었다.
