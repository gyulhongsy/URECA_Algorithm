# 프로그래머스 길 찾기 게임

[문제 링크](https://school.programmers.co.kr/learn/courses/30/lessons/42892)

## 접근
- 주어진 노드 배열을 정렬
  - y좌표 기준(내림차순), y좌표가 같을 시 x좌표 기준(오름차순)
- 노드 클래스를 생성하여 정렬한 맵을 돌면서 각 노드를 연결 (트리 생성)
- 트리를 각 순회에 따라 돌며 정답 저장

### 정답 코드
```
import java.util.ArrayList;
import java.util.TreeMap;

public class Solution {

    static class Node {
        int x; //자식 위치를 결정할 x좌표
        int idx;
        Node left;
        Node right;

        public Node(int x, int idx) {
            this.x = x;
            this.idx = idx;
        }
    } //class - Node

    ArrayList<Integer> list;

    public int[][] solution(int[][] nodeinfo) {
        int[][] answer = new int[2][nodeinfo.length];

        TreeMap<int[], Integer> map = new TreeMap<>((a, b) -> {
            if (a[1] == b[1]) return a[0] - b[0];
            return b[1] - a[1];
        });

        for (int i = 0; i < nodeinfo.length; i++) {
            map.put(nodeinfo[i], i + 1);
        }

        Node parent = null;
        for (int[] key : map.keySet()) {
            int x = key[0];
            int idx = map.get(key);
            Node child = new Node(x, idx);

            if (parent == null) parent = child;
            else insertNode(parent, child);
        } //for - 맵을 순회하며 트리 조합

        list = new ArrayList<>();
        preorder(parent);
        answer[0] = list.stream().mapToInt(i -> i).toArray();

        list = new ArrayList<>();
        postorder(parent);
        answer[1] = list.stream().mapToInt(i -> i).toArray();

        return answer;
    } //Solution

    private void insertNode(Node parent, Node child) {
        if (child.x < parent.x) { //자식의 x좌표가 부모의 x좌표보다 작음
            if (parent.left == null) parent.left = child;
            else insertNode(parent.left, child);
        } else { //자식의 x좌표가 부모의 x좌표보다 큼
            if (parent.right == null) parent.right = child;
            else insertNode(parent.right, child);
        }
    } //insertNode

    private void preorder(Node parent) {
        if (parent == null) return;
        
        list.add(parent.idx);
        preorder(parent.left);
        preorder(parent.right);
    } //preorder

    private void postorder(Node parent) {
        if (parent == null) return;
        
        postorder(parent.left);
        postorder(parent.right);
        list.add(parent.idx);
    } //postorder
}
```

### 회고
- 처음에는 `HashMap`에 노드 데이터를 저장해 정렬하려고 했으나 해시맵은 데이터를 정렬하지 않는 자료구조였다. 따라서 자료를 정렬하기 위해 트리맵을 사용하였다.
- 트리를 만들기 위해 데이터를 정렬하는 부분 까지는 간단하게 생각할 수 있었으나 각 노드를 연결하는 과정을 생각하는게 쉽지 않았다.
- 전위, 후위 순위를 돌 때 탈출 코드를 적어주지 않아 무한 루프에 빠졌었다..
- 내가 작성한 코드를 개선할 방법이 있는지 조금 더 찾아보았다.

## 개선
- 구상한 로직 흐름은 개선할 부분이 없었지만 자료구조의 선택과 객체지향적 설계 면에서 개선할 점이 있었다.
- 정렬 방식
  - 내가 사용한 방식 (TreeMap): 트리맵은 내부적으로 꽤 무거운 트리 구조를 계속 생성하기 때문에 메모리와 시간을 조금 더 씀, 나중에 트리를 만들기 위해 맵에서 다시 꺼내 `Node`로 포장해야 함
  - 개선: 처음부터 `Node` 클래스에 y좌표 정보까지 다 넣고, 입력 받자마자 모든 데이터를 객체 배열 `Node[]`로 만들어 정렬
- 결과 담기
  - 순회 결과를 `ArrayList` 에 담고 `stream().mapToInt().toArray()`로 변환하는 것은 성능 면에서 다소 느림
  - 개선: 노드의 개수를 알고 있으므로 처음부터 정답 배열 `answer[0]` 과 `answer[1]` 을 만들어 두고 순회 메서드 안에서 인덱스 변수를 이용해 배열에 직접 값을 꽂아 넣음
 
### 개선 코드
```
import java.util.Arrays;

public class Solution {

    static class Node {
        int x;
        int y;
        int value;
        Node left;
        Node right;

        public Node(int x, int y, int value) {
            this.x = x;
            this.y = y;
            this.value = value;
        }
    } //class - Node

    int[][] answer;
    int idx; //배열에 값을 넣기 위한 인덱스

    public int[][] solution(int[][] nodeinfo) {
        int n = nodeinfo.length;
        answer = new int[2][n];

        Node[] nodes = new Node[n];
        for (int i = 0; i < n; i++) {
            nodes[i] = new Node(nodeinfo[i][0], nodeinfo[i][1], i+1);
        } //for - Node 객체 배열 생성

        Arrays.sort(nodes, (a, b) -> {
            if (a.y == b.y) return a.x - b.x;
            return b.y - a.y;
        }); //sort
        
        Node parent = nodes[0];
        for (int i = 1; i < n; i++) {
            insertNode(parent, nodes[i]);
        } //for - 트리 구성
        
        idx = 0;
        preorder(parent);
        
        idx = 0;
        postorder(parent);

        return answer;
    } //Solution

    private void insertNode(Node parent, Node child) {
        if (child.x < parent.x) { //자식의 x좌표가 부모의 x좌표보다 작음
            if (parent.left == null) parent.left = child;
            else insertNode(parent.left, child);
        } else { //자식의 x좌표가 부모의 x좌표보다 큼
            if (parent.right == null) parent.right = child;
            else insertNode(parent.right, child);
        }
    } //insertNode

    private void preorder(Node node) {
        if (node == null) return;

        answer[0][idx++] = node.value;
        preorder(node.left);
        preorder(node.right);
    } //preorder

    private void postorder(Node node) {
        if (node == null) return;

        postorder(node.left);
        postorder(node.right);
        answer[1][idx++] = node.value;
    } //postorder
}
```
