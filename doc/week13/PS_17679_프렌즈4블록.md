# 프로그래머스 프렌즈4블록

[문제 링크](https://school.programmers.co.kr/learn/courses/30/lessons/17679)

## 접근
- 주어진 문자열 배열을 2차원 문자 배열로 변환 (blocks)
- matched 배열로 2x2가 완성됐는지 확인
  - 문자 배열을 m-1, n-1 까지 돌면서 2x2 모양이 만들어지는지 확인
  - 맞으면 4자리 모두 matched 배열을 true로 변경, 아니면 넘어가기
- 완성된 matched 배열에서 true에 해당하는 인덱스의 blocks 배열에서 문자를 `-`로 변경 (공백을 의미)
- blocks 배열을 아래부터 돌면서 세로 인덱스만 검사, 공백이 아닌 문자들을 공백 자리로 내림

### 정답 코드
```
public class Solution {
    public int solution(int m, int n, String[] board) {
        int answer = 0;
        char[][] blocks = new char[m][n];

        for (int i = 0; i < m; i++) {
            for (int j = 0; j < n; j++) {
                blocks[i][j] = board[i].charAt(j);
            }
        } //for - insert blocks

        while (true) {
            boolean[][] matched = new boolean[m][n];
            boolean hasPopped = false;

            for (int i = 0; i < m - 1; i++) {
                for (int j = 0; j < n - 1; j++) {
                    char c = blocks[i][j];
                    
                    if (c == '-') continue; //이미 빈칸이면 건너뛰기
                    
                    if (blocks[i][j + 1] == c && blocks[i + 1][j] == c && blocks[i + 1][j + 1] == c) {
                        matched[i][j] = true;
                        matched[i][j + 1] = true;
                        matched[i + 1][j] = true;
                        matched[i + 1][j + 1] = true;
                    } //if - 2x2 찾음
                }
            } //for - 2x2 검사

            for (int i = 0; i < m; i++) {
                for (int j = 0; j < n; j++) {
                    if (matched[i][j]) {
                        answer++;
                        blocks[i][j] = '-';
                        hasPopped = true;
                    }
                }
            } //for - 블럭 터트리기

            if (!hasPopped) break;

            for (int j = 0; j < n; j++) {
                int writeIdx = m - 1; //바닥 위치부터 시작

                for (int i = m - 1; i >= 0; i--) {
                    if (blocks[i][j] != '-') {
                        blocks[writeIdx][j] = blocks[i][j];
                        writeIdx--;
                    } //if - 블록 찾음
                }

                for (int i = writeIdx; i >= 0; i--) {
                    blocks[i][j] = '-';
                } //for - 블록 밀고 남은 부분은 공백으로
            } //for - 빈칸 채우기
        } //while

        return answer;
    }
}
```

### 회고
- 로직은 잘 짰지만 블록을 터트리는 부분에서 빈칸으로 이루어진 2x2인 경우 그냥 넘어가는 조건을 걸지 않아 빈칸도 계속 터트리는 무한 루프에 갇혔었다.
- 이런 사소한 조건도 꼼꼼하게 검사하는 습관을 들여야겠다.
- 공백 자리를 그 위의 블록으로 채우는 로직을 짜기가 어려웠다. 제대로 이해하고 넘어갈 필요가 있었다.
