# 프로그래머스 연속된 부분 수열의 합

[문제 링크](https://school.programmers.co.kr/learn/courses/30/lessons/178870?language=java)

## 접근
- 투 포인터 방식을 사용
- 인덱스를 투 포인터로 설정하고, 합을 구해가며 조절함

### 정답 코드
```
public class Solution {
    public int[] solution(int[] sequence, int k) {
        int[] answer = new int[2];

        int left = 0, right = 0;
        int min = Integer.MAX_VALUE;

        int sum = sequence[0];
        while (left < sequence.length && right < sequence.length) {
            if (sum < k) {
                right++;
                if (right < sequence.length) sum += sequence[right];
            } else if (sum > k) {
                sum -= sequence[left];
                left++;
            } else {
                int diff = right - left;
                if (diff < min) {
                    min = diff;
                    answer[0] = left;
                    answer[1] = right;
                }

                // 정답을 찾았더라도 또 다른 정답이 있는지 찾기 위해 탐색
                sum -= sequence[left];
                left++;
            } //if ~ else
        } //while

        return answer;
    }
}
```
