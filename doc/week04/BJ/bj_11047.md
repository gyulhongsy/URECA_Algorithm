# 백준 11047번 동전 0

## 문제
준규가 가지고 있는 동전은 총 N종류이고, 각각의 동전을 매우 많이 가지고 있다.

동전을 적절히 사용해서 그 가치의 합을 K로 만들려고 한다. 이때 필요한 동전 개수의 최솟값을 구하는 프로그램을 작성하시오.

## 입력
첫째 줄에 N과 K가 주어진다. (1≤N ≤ 10, 1 <K < 100,000,000)

둘째 줄부터 N개의 줄에 동전의 가치 A가 오름차순으로 주어진다. (1<A ≤ 1,000,000, A1 = 1, i ≥ 2인 경우에 A는 A-1의 배수)

## 출력
첫째 줄에 K원을 만드는데 필요한 동전 개수의 최솟값을 출력한다.

## 정답 코드
```
import java.io.BufferedReader;
import java.io.IOException;
import java.io.InputStreamReader;
import java.util.StringTokenizer;

//Greedy
public class Main {
	
	static int N, K;
	static int[] money;
	static int coins = 0;

	public static void main(String[] args) throws IOException {
		BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
		StringTokenizer st = new StringTokenizer(br.readLine(), " ");
		
		N = Integer.parseInt(st.nextToken());
		K = Integer.parseInt(st.nextToken());
		money = new int[N];
		
		for (int i = 0; i < N; i++) {
			money[i] = Integer.parseInt(br.readLine());
		} // for - insert coins
		br.close();
		
		for (int i = N-1; i >= 0; i--) {
			if (money[i] <= K) {
				coins += (K / money[i]);
				K %= money[i];
			}
			if (K <= 0) break;
		} //for
		
		System.out.println(coins);
	} //main
}
```

## 회고
- 그리디 알고리즘의 기본적인 로직이다.
- 하지만 이 경우 동전의 단위가 달라지면 항상 최적해를 보장할 수 없다.
- 최적해 보장이 안되는 예
  ```text
  동전이 100원, 400원, 500원 이렇게 세 가지 있는 경우 800원을 거슬러 줄 때
  그리디 알고리즘을 사용하면 500원 1개, 100원 3개 => 총 4개의 동전
  그러나 400원 동전 2개를 사용하여 거슬러 줄 수 있다.
  ```

# 백준 11047번 동전 0 - 개선 버전
문제의 입력 부분에 있는 `(1<A ≤ 1,000,000, A1 = 1, i ≥ 2인 경우에 A는 A-1의 배수)` 를 활용하여 개선 버전의 알고리즘을 짤 수 있다.

그리디 알고리즘보단 DP(Dynamic Programming)을 사용하여 작성하였다.

## 정답 코드 (java)
```
import java.io.BufferedReader;
import java.io.IOException;
import java.io.InputStreamReader;
import java.util.Arrays;
import java.util.StringTokenizer;

public class Main {

	public static void main(String[] args) throws IOException {
		BufferedReader bufRead = new BufferedReader(new InputStreamReader(System.in));
		StringTokenizer token = new StringTokenizer( bufRead.readLine(), " " );

		int N = Integer.parseInt( token.nextToken() );
		int K = Integer.parseInt( token.nextToken() );

		int [] coins = new int[N];
		for (int i = 0; i < N; i++) {
			coins[i] = Integer.parseInt( bufRead.readLine() );
		} // for

		bufRead.close();
		
		int[] dp = new int[K+1]; //dp[i] = i원을 만들기 위한 최소 동전 개수
		Arrays.fill(dp, Integer.MAX_VALUE); //우선 가장 큰 수로 초기화
		
		dp[0] = 0; //0원 만드는데 동전 0개

    //핵심 로직 부분 예) 동전의 단위가 1원, 4원이 있고
		for (int coin : coins) { //현재 coin = 4 일 때
			for (int i = coin; i <= K; i++) { //i = 6
				dp[i] = Math.min(dp[i], dp[i-coin] + 1); //6원을 만드는 방법 = 2원을 만드는 최소 개수 + 4원 동전 1개
			}
		} //for - dp
		
		System.out.println(dp[K]);
	} // main

} // class
```

## 정리
핵심 점화식: `dp[i] = min(현재값, dp[i-coin] + 1)`

-> i원을 만들려면 현재 coin 하나 쓰고, 나머지 (i-coin)원을 만들면 됨

마지막으로 둘 중 더 작은 값을 최솟값으로 배열에 저장
