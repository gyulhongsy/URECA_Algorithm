# 프로그래머스 석유 시추

[문제 링크](https://school.programmers.co.kr/learn/courses/30/lessons/250136)

## 초기 접근
- 세로로 한 칸씩 내려가며 검사하고 이것을 가로 길이만큼 반복
- 내려갈 때 석유를 만나면 bfs 실행 (dfs 재귀 스택 오버플로우 발생 방지)
- 구한 값을 더함 (한 줄에서 구할 수 있는 석유의 총량)
- 바닥까지 내려갔을 때 더한 석유 값으로 max값 갱신
- 가로 한 칸 옮길 때마다 visited, 석유 값 초기화

### 초기 코드
```
import java.util.ArrayDeque;
import java.util.Queue;

public class Solution {
	
	static int N, M;
	static boolean[][] visited;
	
	static int[] dx = {-1, 1, 0, 0};
	static int[] dy = {0, 0, -1, 1};
	
	static int solution(int[][] land) {
        int answer = Integer.MIN_VALUE;
        
        N = land.length; //세로
        M = land[0].length; //가로
        
        Queue<int[]> que = new ArrayDeque<>();
        
        for (int y = 0; y < M; y++) {
        	visited = new boolean[N][M];
        	int cnt = 0;
        	
        	for (int x = 0; x < N; x++) {
        		if (land[x][y] == 1 && !visited[x][y]) {
        			visited[x][y] = true;
        			cnt++;
        			que.offer(new int[] {x, y});
        			
        			while (!que.isEmpty()) {
        				int[] cur = que.poll();
        				
        				for (int d = 0; d < 4; d++) {
        					int nx = cur[0] + dx[d];
        					int ny = cur[1] + dy[d];
        					
        					if (nx >= 0 && nx < N && ny >= 0 && ny < M
        							&& land[nx][ny] == 1 && !visited[nx][ny]) {
        						visited[nx][ny] = true;
        						cnt++;
        						que.offer(new int[] {nx, ny});
        					} // if - 주변에서 석유 찾음
        				} //for - 상하좌우탐색
        			} //while - bfs
        		} //if 
        	} //for - x
        	
        	answer = Math.max(answer, cnt);
        } //for - y
        
        return answer;
    }
}
```

### 문제점
- 정확성 테스트는 모두 통과했지만 효율성 테스트에서 모두 실패하였다.
- dfs로 바꿔서 제출해봤지만 결과는 똑같았다.
- 세로 한 줄마다 bfs를 실행하는 것이 문제였다. land의 크기가 매우 큰 경우 시간 초과가 발생한다.

## 개선
- bfs는 처음 한 번만 실행하고, 이때 각 좌표별로 구할 수 있는 석유의 양을 저장해둔다.
- 이후에는 land 배열을 돌면서 저장된 석유 양을 바탕으로 구할 수 있는 총 석유의 양만 구한다.

### 개선 코드
```
import java.util.ArrayDeque;
import java.util.HashSet;
import java.util.Queue;
import java.util.Set;

public class Solution {
	
	static int N, M;
	static int[][] group;
	static int[] size;
	
	static int[] dx = {-1, 1, 0, 0};
	static int[] dy = {0, 0, -1, 1};
	
	static int solution(int[][] land) {
        int answer = Integer.MIN_VALUE;
        
        N = land.length; //세로
        M = land[0].length; //가로
        group = new int[N][M];
        size = new int[N * M + 1];
        
        int id = 1;
        
        for (int x = 0; x < N; x++) {
        	for (int y = 0; y < M; y++) {
        		if (land[x][y] == 1 && group[x][y] == 0) {
        			bfs(x, y, id, land);
        			id++;
        		} //if - bfs
        	} //for - y
        } //for - x
        
        for (int y = 0; y < M; y++) {
        	Set<Integer> set = new HashSet<>();
        	
        	for (int x = 0; x < N; x++) {
        		if (group[x][y] != 0) {
        			set.add(group[x][y]);
        		} //if - 몇 번째 석유 덩어리를 만났는지
        	} //for - x
        	
        	int sum = 0;
        	for (int s : set) sum += size[s];
        	
        	answer = Math.max(answer, sum);
        } //for - y
         
        return answer;
    }
	
	private static void bfs(int x, int y, int id, int[][] land) {
		Queue<int[]> que = new ArrayDeque<>();
		
		que.offer(new int[] {x, y});
		group[x][y] = id;
		
		int cnt = 1;
		
		while (!que.isEmpty()) {
			int[] cur = que.poll();
			
			for (int d = 0; d < 4; d++) {
				int nx = cur[0] + dx[d];
				int ny = cur[1] + dy[d];
				
				if (nx >= 0 && nx < N && ny >= 0 && ny < M
						&& land[nx][ny] == 1 && group[nx][ny] == 0) {
					group[nx][ny] = id;
					cnt++;
					
					que.offer(new int[] {nx, ny});
				} //if - 상하좌우탐색
			} //for - d
		} //while
		
		size[id] = cnt;
	} //bfs
}
```

### 회고
- 이번 문제는 bfs로 구현 자체는 비교적 쉽게 했지만, 효율성에서 어려움을 겪었다.
- 이후 석유 덩어리를 한 번만 bfs로 그룹화하고, 그룹 번호와 크기를 저장하는 방식으로 개선했다.
- 열을 탐색할 때는 bfs를 다시 수행하지 않고 set에 만난 석유 덩어리의 번호를 저장하여 그 크기를 합산했다.
- 이번 문제를 통해 알고리즘 구현 뿐만 아니라 탐색 결과를 재사용하여 시간 복잡도를 줄이는 것이 중요하는 점을 배웠다.
