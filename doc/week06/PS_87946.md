# 프로그래머스 피로도

[문제 링크](https://school.programmers.co.kr/learn/courses/30/lessons/87946)

## 접근
- 던전의 수 N개에서 N개를 선택해 던전 순열을 만듦 (던전의 방문 순서)
- 순열 만들고 나면 피로도를 통해 방문한 던전의 수 계산 시작
- 방문한 던전의 수들을 비교하여 max값 구하기

## 정답 코드
```
class Solution {
    
    static int N;
	static int answer = Integer.MIN_VALUE;
	static int[] output;
	static boolean[] visited;

	public int solution(int k, int[][] dungeons) {
        
        N = dungeons.length;
        output = new int[N]; //던전의 인덱스 저장
        visited = new boolean[N];
        
        permunation(0, k, dungeons);
        
        return answer;
    } //solution

	private void permunation(int idx, int k, int[][] dungeons) {
		if (idx >= N) {
			int cnt = 0;
			for (int x : output) {
				if (k > 0) {
					if (k >= dungeons[x][0]) {
						k -= dungeons[x][1];
						cnt++;
					} else break;
				} // if - cnt
			} // for - 던전 탐방
			answer = Math.max(answer, cnt);
		} //if - base case
		
		for (int i = 0; i < N; i++) {
			if (visited[i]) continue;
			
			visited[i] = true;
			output[idx] = i;
			permunation(idx+1, k, dungeons);
			visited[i] = false;
		} //for - recursion
	} //permutation
}
```

## 회고
- 순열을 생성한 뒤 완전 탐색으로 모든 경우의 수를 계산하였다.
- 프로그래머스 통과에 문제는 없었지만 순열을 만드는 중간에 피로도를 검사하면 조금 더 효율적으로 탐색할 수 있을 것 같다.

## 개선 코드
```
class Solution {

    static int answer = 0;
    static boolean[] visited;

    public int solution(int k, int[][] dungeons) {
        visited = new boolean[dungeons.length];

        dfs(k, 0, dungeons);

        return answer;
    } //solution

    static void dfs(int k, int cnt, int[][] dungeons) {
        answer = Math.max(answer, cnt);

        for (int i = 0; i < dungeons.length; i++) {
            if (visited[i]) continue;

            if (k < dungeons[i][0]) continue;

            visited[i] = true;
            dfs(k-dungeons[i][1], cnt+1, dungeons);
            visited[i] = false;
        } //for - recursion
    } //dfs
}
```
